# Permit to Work (PTW) Web Application

This project is a Permit to Work (PTW) web application designed specifically for ship operations.

## Features
- **PTW Creation/Editing**: Easily create and manage permits.
- **QR Code Scanning**: Scan codes to streamline operations.
- **PDF Printing**: Generate printable PDFs of permits.
- **Activity Logging**: Keep track of all PTW activities.

This application aims to improve safety and efficiency in ship operations by providing a comprehensive digital solution for managing permits.
